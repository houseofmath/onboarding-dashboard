# Onboarding Response Dashboard

Interactive web dashboard for exploring onboarding response patterns by country, role, and conversion group (based on data collected from 2026.02.10 to 2026.03.31).

## Features
- Subscriber vs. signed-up vs. not-signed-up comparison
- Full response distribution by step
- NO vs US comparison
- Canonical entity-level onboarding analysis

## Deployment
This repository is deployed as a static webpage for easy access and sharing.

View the dashboard here: [https://houseofmath.github.io/onboarding-dashboard/](https://houseofmath.github.io/onboarding-dashboard/)
